API Endpoints

POST /api/deploy

Initial application deployment request

{
  "email": "student@example.com",
  "secret": "your-secret",
  "task": "unique-task-id",
  "round": 1,
  "nonce": "unique-nonce",
  "brief": "Application description",
  "checks": ["Requirement 1", "Requirement 2"],
  "evaluation_url": "https://webhook.site/url",
  "attachments": []
}
POST /api/revise

Application revision and update request

{
  "email": "student@example.com",
  "secret": "your-secret",
  "task": "unique-task-id",
  "round": 2,
  "nonce": "unique-nonce",

  "checks": ["New requirement 1", "New requirement 2"],

  "attachments": []
}

Check deployment status

GET /health

Health check and system status
